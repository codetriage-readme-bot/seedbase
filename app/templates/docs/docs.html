The API

The API is a JSON API and XML is not supported. Endpoints are documented with the HTTP method for the request and a partial resource identifier. Curly braces, {}, indicate values you have to supply. For example:

The examples in the documentation are cURL statements. You can run the statements on a command line to try out different API requests. To learn more, see the cURL website.

Authentication

Basic authentication is supported with an email address and password. Use the following authentication format with your email address and password:

{email_address}:{password}

Rate Limits

The API has a rate limit of 1,000 requests/month.

Request Format

Since the Seedbase Data API is a JSON API, you must supply a Content-Type: application/json header in PUT and POST requests and set an Accept: application/json header on all requests.

Response Format

Successful responses will be in the 200 or 300 range.
